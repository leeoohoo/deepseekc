# DeepSeek CLI Electron UI SVG 设计规范

## 颜色主题

### 背景渐变
- 主背景：线性渐变从上到下 `#0f172a` (dark-900) 到 `#1e293b` (dark-800)
- 备选背景：`#1e293b` (dark-800)

### 颜色调色板
- 背景色：`#0f172a` (dark-900), `#1e293b` (dark-800), `#334155` (dark-700)
- 文字颜色：
  - 主要文字：`#f1f5f9` (dark-100)
  - 次要文字：`#94a3b8` (dark-400)
  - 标签文字：`#6366f1` (primary-500)
- 边框颜色：
  - 默认边框：`#475569` (dark-600)
  - 高亮边框：`#6366f1` (primary-500)
- 高亮颜色：
  - 主高亮：`#6366f1` (primary-500)
  - 次要高亮：`#818cf8` (primary-400)
- 状态颜色：
  - 成功：`#10b981` (green-500)
  - 警告：`#f59e0b` (amber-500)
  - 错误：`#ef4444` (red-500)
  - 信息：`#3b82f6` (blue-500)

## 字体
- 主要字体：Inter (sans-serif)
- 等宽字体：JetBrains Mono (monospace)
- 字体大小：
  - 标题：14px
  - 正文：12px
  - 标签：11px
  - 代码：11px

## 间距与尺寸
- SVG 尺寸：800px × 500px
- 内边距：20px
- 边框圆角：8px
- 元素间距：10px

## 标注样式
- 标注框：背景色 `#334155` (dark-700)，边框 `#475569` (dark-600)，圆角 6px，内边距 8px
- 标注箭头：颜色 `#6366f1` (primary-500)
- 标注文字：颜色 `#f1f5f9` (dark-100)，字体大小 11px，加粗

## 组件样式

### 按钮
- 主要按钮：背景 `#6366f1`，文字白色，圆角 6px
- 次要按钮：背景 `#334155`，边框 `#475569`，文字 `#f1f5f9`，圆角 6px

### 输入框
- 背景：`#1e293b` (dark-800)
- 边框：`#475569` (dark-600)
- 文字：`#f1f5f9` (dark-100)
- 圆角：6px

### 面板
- 背景：`#1e293b` (dark-800) 带 0.8 透明度
- 边框：`#475569` (dark-600)
- 圆角：8px
- 阴影：轻微内阴影

### 代码高亮
- 背景：`#0f172a` (dark-900)
- 文字：`#f1f5f9` (dark-100)
- 关键字：`#818cf8` (primary-400)
- 字符串：`#10b981` (green-500)
- 注释：`#64748b` (dark-500)

## 设计原则
1. 扁平化设计：无渐变、无阴影（除非必要）
2. 简洁明了：只展示核心功能区域
3. 一致性：所有 SVG 使用相同的颜色、字体和间距
4. 可读性：标注清晰，文字大小合适
5. 专业感：对齐整齐，颜色协调
